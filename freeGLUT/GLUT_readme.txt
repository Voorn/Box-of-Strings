This is a compilation of freeglut for Windows, an opensource alternative to GLUT.
For more information, see: https://freeglut.sourceforge.net/

The windows binary might not work for you immediately due to not having OpenGL, for instance you might get the error "GLUTinit" not found. If so, move the glut32.dll file to the same folder as the binary. You could also use the link above to compile your own version of glut.
